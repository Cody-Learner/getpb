# getpb<br>

Getpb is an Arch package information gathering tool for official and AUR packages. 	<br>
											<br>
Depends: base-devel asp expac auracle							<br>
											<br>
Usage: getpb [Operation] \<package-name\>						<br>
											<br>
 Operations:										<br>
	-p  = Print PKGBUILD								<br>
	-s  = Print .SRCINFO								<br>
	-m  = List makedepends								<br>
	-h  = Help									<br>
